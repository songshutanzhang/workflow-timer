# 计时器 ⏱

为流程性工作设计的极简网页计时器：**倒计时** / **秒表** / **任务**

## ✨ 核心功能

- **三种模式**：
  - **倒计时**：一次性倒计时，输入时长直接开始，多快捷时长可选，支持自定义快捷时长。
  - **秒表**：精确到厘秒级(0.01s)，支持分段记录，支持导出为CSV/Excel。
  - **任务**：按序执行多个子任务倒计时，适用于流程性工作。
- **提前/延后**：可提前进入下一流程，子任务到点不强制结束，可继续累计超出时间。
- **总览对比**：随时查看计划总时长 vs 实际用时 vs 差异。支持导出为CSV/Excel。
- **极简 UI**：浅色/深色主题，可隐藏边栏，可离线使用。
- **全屏**：原生支持浏览器全屏，一键进入，专注模式。

## 🚀 快速开始

1. 解压 `workflow-timer.zip` 到任意目录
2. 点击 `index.html`
3. 选择用浏览器打开（推荐 Edge / Chrome ）

> 退出全屏：`Esc`/`F`/`F11`

## 📋 三种模式说明

### 1. 倒计时

- 在中栏输入 `时:分:秒`或选择快捷时长
- 点 `▶ 开始` 倒计时
- 归零后变红色提示音

### 2. 秒表

- 点 `▶ 开始` 从 0 累加
- 可点 `⊕` 或 `L` 分段记录
- 支持厘秒精度（0.01s）
- 支持导出为CSV/Excel格式

### 3. 任务

适合：考试 / 会议流程 / 多阶段工作

#### 使用流程

**步骤 1：新建任务组**

点击边栏"新建任务组"，设定名字并添加子任务

示例：

```
四六级
  ├ 写作  30:00
  ├ 听力  30:00
  ├ 阅读  40:00
  └ 翻译  30:00

工作会议
  ├ 准备材料  10:00
  └ 汇报       5:00
```

**步骤 2：启动**

- 点组旁的 `▶ 开始` 按钮：启动整组，按序执行。
- 也可选择边栏中任意子任务，从任意子任务开始执行。

**步骤 3：进行中**

- 顶部出现橙色"任务组"横幅，显示进行/暂停状态
- 中栏显示当前项 + 倒计时
- 边栏实时显示每个子任务的实际用时
  - 未开始：`--:--`
  - 进行中：实时累计（蓝色）
  - 已完成：实际用时（绿色）
- 底部显示计划用时，实际用时与差异

**步骤 4：切换任务**

- 点 `⏭ 下一项`，或按`→`/ `N`
- 立即切到下一项
- 点击 `继续` 继续计时

**步骤 5：完成**

- 点 `结束全部` 结束任务组

## 🔧 技术细节

- **零依赖**：纯 HTML + CSS + 原生 JS，单文件运行
- **数据存储**：浏览器 localStorage（无数据上传）
- **实际用时精度**：`performance.now()` 时间戳差值（毫秒精度）
- **自适应**：自适应窗口（推荐 1280×720 以上）

## ❓ 常见问题

**Q: 数据存在哪？**

A: 浏览器 localStorage 位于本地，不会有泄露风险。如果换浏览器或清缓存会丢失。

**Q: 时间到 0 后还在跑？**

A: 这是有意设计的"延后模式"——工作中常有某阶段超时的情况，需要继续累计。等你切换其它子任务才会切走。

**Q: 停止任务组后，再启动之前的计时还保留吗？**

A: 如果是暂停，退出重进后原有的计时仍保留。结束任务组后不会保留。
